# gvard.github.io/web/

Educational web pages.

Простой список:
```html
  <nav>
    <ul>
      <li>
        <a href="http://">Ссылка 1</a>
      </li>
      <li><a href="http://">Ссылка 2</a></li>
      <li><a href="http://">Ссылка 3</a></li>
    </ul>
  </nav>
```

```css
nav ul {
  background-color: blue;
}
nav ul li {
	display:inline-block;
}
nav a {
	display: block;
	padding: 0 10px;
	color: white;
	font-size: 16px;
	line-height: 40px;
	text-decoration: none;
}
nav a:hover {
  background-color: #566DD6;
}
```
