Инструменты

Для того, чтобы создавать и просматривать web-страницы, нужно всего две программы: текстовый редактор и web-браузер. Хорошим выбором будет Microsoft Visual Studio Code (есть поддержка русского языка в интерфейсе) и Google Chrome. Если VSCode ощутимо "тормозит", подойдет более простой редактор Notepad++. В качестве браузера также подойдут Apple Safari, Mozilla Firefox и Microsoft Edge. Использование Microsoft Internet Explorer нежелательно, так как он устарел и не поддерживает должным образом многие современные технологии.

В текстовом редакторе создаются файлы с расширением html и css. Их необходимо сохранить на диске. Затем файл html открывается в браузере при помощи диалогового окна, вызываемого нажатием клавиш Ctrl+O. Для просмотра исходного кода страницы следует нажать Ctrl+U. Визуальный инспектор кода вызывается сочетанием клавиш Ctrl+Shift+C. Консоль разработчика доступна при нажатии F12.

В дальнейшем файлы web-проекта могут быть выложены на хостинг для доступа к ним по сети Интернет через URL, уникальный адрес. Хостинг может быть бесплатным, как, например, 000webhost.com. После регистрации становится возможным выкладывать свои web-страницы в открытый доступ.

Для быстрой проверки кода подойдут онлайн-сервисы типа Codepen.io. После регистрации своими примерами можно делиться!
В окно HTML вставляется код из тега <body>. В окно CSS копируется содержимое стилевого файла и/или содержимое html-тега <style>. Окно JavaScript можно скрыть за недадобностью.

Основы HTML

Web-страницы созаются на языке гипертекствой разметки, HyperText Markup Language. В HTML для разметки пользуются тегами – ключевыми словами в угловых скобках. Теги бывают одиночные и парные. Парные теги являются контейнерами, в которых можно расположить текст, изображения или иной объект. Web-дизайнер должен понимать смысл и назначение основных тегов html.

Как сделать карусель

Учебные страницы к занятиям:

  • Web-кот и его Собака. Задача: создать html-страницу с элементами заголовок, параграф, изображение, список, ссылка и открыть ее в браузере.
  • Домашняя страница Кота. Задача: создать страницу с оформлением из данного примера. Создать и подключить стилевой файл, вставить иконку страницы, изменить фон страницы, поместить на фон картинку.
  • Как выглядит семантическая разметка. Памятка по структуре документа.
  • Простая статья с оформлением. Задача: создать html-страницу с семантическими тегами header, main, footer, article. Заполнить тег article статьей. Выделить основное содержимое средствами css.
  • Новостная страница с меню. Задача: создать страницу с семантической версткой. В шапку сайта вставить меню, сделанное на основе ненумерованного списка. Оформить страницу по образцу.
