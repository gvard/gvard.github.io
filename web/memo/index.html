Автоматизируем Web-разработку: памятка

Начало работы

  1. Создать папку (каталог, директорию), где будут лежать все файлы проекта
  2. Создать первый файл проекта в текстовом редакторе с функцией автодополнения кода (сочетание клавиш Ctrl+N). Подойдет свободный редактор Microsoft Visual Studio Code
  3. Сохранить (Ctrl+S) новый пустой файл в созданной папке с именем index и задать ему расширение html (php). Полное имя файла – index.html (index.php)
  4. В первой строке нового файла набрать ! (восклицательный знак, Shift+1) и нажать ⏎ Enter. Появится первичная разметка HTML-документа. Так работает автодополнение кода Emmet. Обрати внимание: курсор стоит на выденении слов device-width. Эту строку трогать не надо!
  5. Для наполнения HTML-документа необходимо переместить курсор на пустую строку в теле документа, между открывающим и закрывающим тегами (элементами) <body> и </body> и начать наполнять документ
  6. Сохранить (Ctrl+S) в документе все сделанные изменения. При включенной опции «Автосохранение» этот шаг пропустить
  7. Открыть браузер или сделать активным (Alt+Tab) окно браузера. Открыть (Ctrl+O) или обновить (Ctrl+R) страницу
