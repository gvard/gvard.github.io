Задание к 30 марта: шрифты

Подготовить (создать, либо изменить существующую) web-страницу с учетом пересланного материала

  • На странице при помощи css должны быть применены ​​свойство font-family (по крайней мере, должно быть 2 семейства шрифтов), font-size, должны быть заголовок и параграф
  • Подключен внешний шрифт
  • Должны быть применены правила переноса текста на новую строку (тегом <pre> или свойством white-space: pre)

Примеры случаев, когда в дизайне используются правила переноса текста и разные семейства шрифтов:

  • Стихи (курсивный шрифт)
  • Примеры кода на языке программирования (моноширинный шрифт)

Задание к 6 апреля: свойства текста

Подготовить (создать, либо изменить существующую) web-страницу с учетом пересланного материала

  • На странице при помощи css должны быть применены ​​свойства text-align, text-align-last, text-indent, line-height и text-decoration
  • Должен быть заголовок
  • Текст должен располагаться в параграфах

Задание к 13 апреля: свойства изображений

Подготовить (создать, либо изменить существующую) web-страницу с учетом пересланного материала

  • На странице при помощи css должны быть применены свойства фона background-position, background-size, background-repeat (или в сокращенной форме background)
  • Должен быть заголовок
  • Должно быть изображение на фоне и картинка поверх фона в рамке
