Использование внешних шрифтов

Для выбора и подключения внешнего шрифта, как правило, используют сайт fonts.google.com. На нем свободно доступно более 900 шрифтов.

Для выбора шрифта нужно нажать на знак добавления "+" в верхнем правом углу блока шрифта. Появится черная полоса снизу с указанием количества выбранных шрифтов. По клику полоса раскрывается, показывая два блока кода.

  • Первый блок — тег <link>. Его необходимо вставить в "голову" (тег <head>) html-файла.
  • Второй блок – правила css для использования выбранных шрифтов. Их вставляют в блок объявлений нужного селектора. На этой странице шрифты применены к параграфам и заголовку <h1>.

Для сохранения переносов строк используется тег <pre>. По умолчанию, для него установлен моноширинный шрифт. Ниже для блока pre переопределены шрифты.
