Страницы, сделанные на занятиях кружков Веб-дизайн

Текущие задания (сентябрь)

  1. Создать (рассказать) историю
  2. Сделать html-страницу, содержащую минимум заголовок (теги h1, h2) и 2 параграфа (тег p): историю и указание автора
  3. Добавить минимум одну иллюстрацию (тег img)
  4. Добавить внутреннюю и внешнюю ссылки (теги a и a:blank)
  5. Сделать несколько страниц и связать их перекрестными ссылками
  6. Проверить страницу при помощи валидатора и исправить все ошибки разметки
  7. Зарегистрироваться на хостинге, выбрать URL-адрес сайта и создать его. Выложить файлы проекта на хостинг.
    Главная страница должна быть в директории public_html и иметь название index.html
  8. Что будет дальше? Добавим на сайт из нескольких страниц шапку и подвал, в шапке сделаем одинаковую для всех страниц навигацию, оформим сайт при помощи каскадных листов стилей css
