CSS-препроцессоры

  • Препроцессор – программа, принимающая данные на входе и выдающая на выходе данные, предназначенные для входа другой программы.
  • Less (Leaner Style Sheets, компактные таблицы стилей) – динамический язык стилей.
  • Less – это вложенный метаязык (nested metalanguage): валидный CSS будет валидным less-кодом с аналогичной семантикой.
  • Less как программа (lessc) – препроцессор, преобразующий код less в валидный css-код.
  • Устанавливается в node.js командой
    npm install less -g
