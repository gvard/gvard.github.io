Astrodata

Астрономические данные: сбор, обработка и визуализация астрономических данных. Серия видео и статей.

Spectractor

Библиотека для обработки астрономических спектральных данных. В библиотеке реализованы функции чтения и записи форматов данных 100, 200 и fits, использующихся комплексом программ DECH

Репозиторий на GitHub, репозиторий на SourceForge

Зависимости

  • Python 2. Последняя версия: 20.04.2020
  • Библиотеки NumPy и Matplotlib
  • PyFITS. Последняя версия: 3.5 от 18.12.2017. Весь функционал PyFITS теперь доступен в Astropy в виде пакета astropy.io.fits
  • PyMidas (для модуля midworker). Последняя версия: 1.0.5 от 06.03.2006
